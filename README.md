# Coding Standard

Not a hard limit by any means, but it should give a good idea

* PEP8, PEP8, PEP8
* Asserts, lots of them
* Right Margin [76]
* Spaces [4]
* Function Line [10]
* Objects when possible
* Descriptive variable names
